# 🌐 Portfolio Website with CMS

A full-stack portfolio platform built with the **MERN stack (MongoDB, Express, React, Node.js)**.  
This application transforms a traditional static portfolio into a **dynamic, content-managed system**, allowing real-time updates through a custom-built admin dashboard. Visit my portfolio website here: [Live Demo](https://mafika-mahlobo.up.railway.app/)


## 🚀 Overview

This project is designed to showcase projects, skills, and personal information while providing a **secure admin interface** to manage all content without touching the codebase.

It includes:

- 🔐 Authentication & protected routes
- 📦 RESTful API for content management
- 🖼️ File uploads (images/media)
- 🧠 Dynamic rendering of portfolio content
- ⚙️ Admin dashboard for CRUD operations



## 🧩 Tech Stack

### Frontend
- React
- Redux Toolkit (state management)
- Tailwind CSS (UI styling)
- Axios (API requests)
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose ODM)
- JWT Authentication
- Multer (file uploads)
- Cloudinary (file storage & CDN)

## 🔑 Features

### 👤 Public Portfolio
- View projects and details
- Responsive UI
- Dynamic content rendering from database

### 🛠️ Admin Dashboard
- Add, edit, and delete projects
- Upload images/files
- Manage portfolio content in real-time

### 🔐 Authentication
- Secure login system (JWT-based)
- Protected admin routes

### 📡 API
- Fully RESTful backend
- CRUD operations for projects and user data
